TAB 2 — Simulate (Corner Frequency Models)

Eq.1 — Brune (1970) Natural Whole-Fault fc

fc = 4.9×10⁶ × β × (Δσ/M₀)^(1/3) = 0.05453 Hz PHYSICAL — observable in recorded seismic spectra. 4.9×10⁶ = constant from Brune circular dislocation model.

Eq.2 — Type I Static fc (FINSIM, Beresnev & Atkinson 1997)

fc_sub = 4.9×10⁶ × β × (Δσ/M₀_sub)^(1/3) where M₀_sub = M₀/N = 4.9×10⁶ × 3.8 × (100/5.69e+25)^(1/3) ⚠️ GRID DEPENDENT — fc changes with N, non-physical for Mw>5.5
N = total subfaults = NL×NW = 70; M₀_sub = M₀/N = 5.69e+25 dyne-cm

Eq.3 — Type II Dynamic fc (EXSIM, Motazedian & Atkinson 2005)

f₀_ij(t) = 4.9×10⁶ × β × [Δσ/(NR(t)·M₀,avg)]^(1/3) [Hz] NR(t) = number of subfaults ruptured at time t M₀,avg = M₀/N = constant At t=0: NR=1 → fc is maximum; At t=end: NR=N → fc = Brune fc ✅ GRID INDEPENDENT — converges regardless of N.
NR(t) = cumulative subfaults ruptured [1 → N]; M₀,avg = 5.69e+25 dyne-cm

Eq.4 — Type III Double-Corner (Atkinson & Silva 2000)

S(f) = (Cr·M₀·f²) × [ε·B(f,fa) + (1-ε)·B(f,fb)] B(f,fx) = 1/(1+(f/fx)²) [Brune shape at corner fx] fa = 0.20·fc = 0.01091 Hz (global fault dimension) fb = 3.50·fc = 0.19086 Hz (asperity scale) ε = 0.55 (weighting) → Produces spectral sag at 0.2–1 Hz seen in large event recordings.

Eq.5 — Type IV Tang (2022), BSSA 112:921

f₀_ij = k · Vr · L̄_ij⁻¹ where L̄_ij = √(NR·Δl·Δw/π) f₀ = k·Vr·√[π/(NR·Δl·Δw)] k = 0.5 (calibration constant); Vr = 3.04 km/s = 0.8β Δl = 7.5 km; Δw = 5 km → Explicit rupture velocity coupling missing from EXSIM.

Eq.6 — Complete SFF Fourier Acceleration Spectrum

|A(f)| = Cr · M₀ · [f²/(fc²+f²)] · G(R) · P(f,R) · K(f) · AF [cm/s] Cr = (Rθφ·Fs·Fc) / (4π·ρ·β³) [radiation constant] = (0.55×2.0) / (4π×2.8×(3.80e+05)³) G(R) = geometric spreading [CGS, 1/cm]: 1/R / 1e5 (R < 40 km) 40⁻¹·(R/40)⁻⁰·⁵ / 1e5 (40 ≤ R ≤ 80 km) 40⁻¹·2⁻⁰·⁵·(R/80)⁻⁰·⁵⁵ / 1e5 (R > 80 km) [Table 4.2, Chopra 2010] P(f,R) = exp(−πfR/Q(f)β) [anelastic attenuation] Q(f) = Q₀·f^η = 790·f^0.35 [Chopra 2010] K(f) = exp(−πκ₀f) = exp(−π×0.0237×f) [kappa filter, Anderson & Hough 1984] AF = (760/Vs30)^0.3 = (760/400)^0.3 = 1.2123 [Vs30 amp.]
Rθφ=0.55 (rms radiation); Fs=2.0 (free surface); ρ=2.8 g/cm³; β=3.8 km/s; κ₀=0.0237 s

TAB 2 — Stochastic Synthesis

Eq.7 — Boore (2003) Stochastic Time History [PGA FIX]

STOCHASTIC SYNTHESIS STEPS: 1. Compute |A(f)| for chosen model [Eq.6] 2. Generate white Gaussian noise n(t), apply Saragoni-Hart taper window w(t) 3. FFT of windowed noise → extract phases φ(f) = arg[FFT(n·w)] 4. Shape with target amplitudes: X(f) = |A(f)| · exp(iφ(f)) 5. Inverse FFT → acc(t), apply window again KEY FIX — RMS normalisation (Boore 2003, Eq.3): σ²_target = (2/T_dur) × ∫|A(f)|² df [one-sided PSD] σ_target = √(2·∫|A(f)|²·df / T_dur) PREVIOUS WRONG: σ = √(2·∫|A(f)|²·df) → 10–20× too large (PGA~1600 cm/s²) CORRECTED: σ = √(2·∫|A(f)|²·df / T_dur) → PGA~50–200 cm/s² ✓ T_dur = T_src + T_path + R/β = 18.3 + 40.4 + 60.5 = 119.3 s

TAB 3–7 — Response Spectrum

Eq.8 — Newmark Average Acceleration (Chopra 2012, §5.2)

SDOF equation: mü + cü + ku = −m·üg(t) For unit mass: ü + 2ζωu̇ + ω²u = −üg(t) Effective stiffness (γ=0.5, β_nb=0.25): k_eff = ω² + 4ζω/Δt + 4/Δt² Modified incremental force: ΔP̂ = Δp + (4/Δt + 4ζω)·v + 2·a_sdof Increments: Δu = ΔP̂ / k_eff Δv = (2/Δt)·Δu − 2v Δa = (4/Δt²)·Δu − (4/Δt)·v − 2·a_sdof PSA(T,ζ) = ω² × max|u(t)| [g] [Reference: Chopra A.K. (2012). Dynamics of Structures. Table 5.2.2] [Also: ASCE 4-16, IEEE 693, IS 1893:2016, IAEA-TECDOC-1333]
ω=2π/T [rad/s]; ζ=0.05 (5% damping); Δt=0.005 s; PSA=pseudo-spectral acceleration
